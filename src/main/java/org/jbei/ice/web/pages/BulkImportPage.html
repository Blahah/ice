Some hints:

  • Warning: Hitting the back button will lose your work.
  • Cells can be automatically copy-filled by grabbing the bottom right corner (a la Excel).
  • If you prefer to edit your import form in Excel, a template Excel file with the appropriate headers are provided here: Template
  • Paste into the bulk import sheet by selecting the top left cell, then right clicking for the paste menu item. Ctrl-V will not work correctly.
  • After submitting, an administrator must approve your submission before it will show up in the search listings. Contact them if you are in a hurry.
