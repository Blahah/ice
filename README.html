ICE Documentation¶

This is the source code of the ICE documentation site. We welcome patches for any typos or errors you find.

Getting Started¶

You’ll need to have Python installed.

  1. Install the Sphinx documentation generator.
pip install sphinx
  1. Check out the ice-docs repository.
git clone https://github.com/JBEI/ice-docs.git
  1. Make your desired changes to index.html or to any of the .rst files referenced in master.rst.
  2. Build the changes with
make html
  1. If all goes well, you’ll be able to view the updated documentation in the _build/html directory.
